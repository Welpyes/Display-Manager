# Display-Manager
Pseudo Display Manager mainly designed for termux

config structure. file is `$HOME/.dmrc`
```ini
[user1]
username= Welpyes
pwd= asdf
cmd= echo "Logged in to Welpyes"
[user2]
username= Bob
pwd= 12345
cmd= echo "Logged in to Bob"
[user3]
username= Guest
pwd= qwerty
cmd= echo "Logged in to Guest"
```
